# Más Café — Web

Sitio web oficial de **Más Café**, marca de café de especialidad y hospitalidad consciente en Cali, Colombia.

Inspirado en la experiencia de marcas como [Tropicalia Coffee](https://www.tropicaliacoffee.com/) y [Pergamino](https://co.pergamino.co/), con la identidad visual y estrategia de marca propias de Más Café.

## Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm run start` | Servidor de producción |
| `npm run lint` | ESLint |

## Estructura

```
src/
├── app/          # Páginas (/, /cafe, /menu, /nosotros, /tienda, /blog, /contacto)
├── components/   # Componentes reutilizables
└── lib/data.ts   # Contenido y datos de marca
public/
├── fonts/        # Satoshi, Playfair Display, Marydale
└── images/       # Logos, ilustraciones y productos
```
